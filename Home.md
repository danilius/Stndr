# Stndr

Stndr is a desktop reader for Jewish texts, inspired by the Yiddish word "Shtender." It helps you build a local study library from Sefaria texts, read Hebrew and translations in a focused workspace, and follow commentaries, linked sources, and search questions without leaving the app.

Stndr is currently an alpha application. The current release is **v0.0.3-alpha**.

## Install

Download the current Windows installer from the [Stndr releases page](https://github.com/danilius/Stndr/releases/tag/v0.0.3-alpha).

After installing, Stndr can check for future updates and show an update banner inside the app. Release packages are Windows-focused for now.

## What Stndr Does

- Browse the Sefaria library by category and book.
- Download Hebrew texts and translations into a Stndr Data folder.
- Reopen downloaded texts without re-downloading them.
- Read Hebrew, translation, or both together.
- Keep multiple reader tabs open during a learning session.
- Jump through chapters, pages, sections, sedrot, aliyot, and other navigation structures where supported.
- View commentaries and linked sources for the selected passage.
- Search installed texts locally, or use Sefaria search online from inside Stndr.
- Customize fonts, reader widths, title language, and local data storage.

## Cached Reading

Stndr is designed around cached reading. Texts and translations that you download are stored locally in your Stndr Data folder and can be opened again later without downloading them again.

Some features still need an internet connection, especially first-time downloads, update checks, Sefaria metadata, commentary lookups, linked-text previews, and online Sefaria search.

## Main Areas

### Library Manager

Use the Library Manager to browse Sefaria, inspect book and category details, choose versions, download texts, and remove local downloads when you no longer need them.

### Reader

Downloaded books open in tabs. Stndr restores open tabs, selected texts, layout, and reading position between sessions, so a study session can continue where you left off.

### Navigation

Navigation adapts to the selected work. Stndr supports schema-aware navigation for Tanach and Bavli, topic-based navigation for works such as Shulchan Aruch, a jump box for available markers, and Chumash navigation by sedra and aliyah.

### Commentaries And Links

Select a passage to load related commentaries and linked sources from Sefaria. Linked sources can be previewed, opened in a new tab, shown in a split view, or downloaded when they are not installed yet.

### Advanced Search

Advanced Search supports reusable study searches across installed books, including word and letter searches, proximity searches, same-segment or same-chapter searches, exclusion terms, saved searches, and optional online Sefaria search.
